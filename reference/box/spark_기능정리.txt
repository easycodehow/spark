====================================
 SPARK 메모앱 - 기능 정리
====================================
작성일: 2026-07-17
기준 문서: CLAUDE.md (spark 메모앱 개발가이드)

------------------------------------
1. 현재 구현된 기능 (CLAUDE.md 지시 기준)
------------------------------------

[기본 CRUD & 저장]
- 메모 생성 / 조회 / 수정 / 삭제
- LocalStorage에 메모 저장 (브라우저 종료 후에도 유지)
- 메모마다 고유 ID 부여 (crypto.randomUUID())
- 실시간 키워드 검색
- 중요 메모(별표) 필터

[UI]
- 메모 입력 영역 + 저장 버튼
- 중요 메모 토글(별 버튼)
- 상세보기 (수정/삭제/공유/복사)
- 더보기 메뉴(⋮): 내보내기/가져오기, 글자크기 조절
- 다크모드 토글 (헤더 우측 아이콘)
- 라이트/다크 컬러 시스템, Pretendard 폰트

[PWA 핵심 기능]
- Web App Manifest
- Service Worker (오프라인 캐싱)
- 오프라인 상태에서 메모 CRUD 정상 동작

[데이터 관리]
- 메모 JSON 내보내기 / 가져오기 (백업 · 복원)
- 이미지 첨부 (Base64, 1장당 최대 500KB 제한)
- LocalStorage 잔여 용량 체크

[고급 PWA 기능]
- 앱 설치 유도 배너 (beforeinstallprompt) → 화면의 "설치" 버튼
- 진동 피드백 (Vibration API) - 저장/삭제 시
- 화면 꺼짐 방지 (Wake Lock API) - 입력 중

[폴더 관리 - 8단계]
- 메모 첫 줄 "폴더명/" 형식 입력 시 자동으로 폴더 그룹화
  (예: "할일/", "살것/", "책/")
- 폴더명 줄은 목록 미리보기에서 숨김 처리
- 상단 폴더 탭 자동 생성 ("전체" + 폴더별 탭 + "미분류")
- 폴더 탭 클릭 시 해당 메모만 필터링
- 폴더 길게 눌러 삭제 → 삭제 시 메모는 "미분류"로 이동 (메모 자체는 삭제 안 함)

[배포]
- Vercel 배포 완료 (https://spark-y9p6.vercel.app/)


------------------------------------
2. 추가로 구현 가능한 PWA 기능 (미구현)
------------------------------------

[PWA 심화]
- 백그라운드 동기화 (Background Sync API)
  : 오프라인 중 변경사항을 온라인 복귀 시 자동 반영
- 푸시 알림 (Push API) : 메모 리마인더 등
- IndexedDB 전환
  : LocalStorage는 용량 제한(5~10MB)이 있어 이미지 많아지면 한계 →
    대용량 저장에는 IndexedDB가 더 적합
- App Shortcuts (manifest shortcuts 필드)
  : 홈 화면 아이콘 길게 눌러 "새 메모 작성" 바로가기
- Share Target API
  : 다른 앱에서 "공유하기"로 텍스트/이미지를 spark로 바로 전달
- Periodic Background Sync : 주기적 자동 백업

[검색 / 정리 강화]
- 다중 태그 시스템 (폴더 외 추가 분류)
- 정렬 옵션 (최신순 / 오래된순 / 제목순)
- 메모 고정(pin) 기능
- 휴지통(soft delete) : 삭제 후 일정 기간 복구 가능

[입력 / 편집 경험]
- 마크다운 간단 서식 지원 (굵게, 목록 등)
- 음성 입력 (Web Speech API)
- 자동 저장 (타이핑 중 주기적 저장, 강제종료 대비)
- Undo / Redo

[보안]
- 앱 잠금 (PIN / 생체인증 - WebAuthn)
- 메모별 암호화 저장 옵션

[동기화 / 백업 확장]
- 클라우드 백업 연동 (Google Drive, iCloud 등)
- 자동 백업 스케줄링

[접근성 / 품질]
- 키보드 단축키 (새 메모, 검색 포커스 등)
- 스크린리더 대응 (ARIA)
- 줄간격 조절 옵션 (현재는 글자크기만 지원)
