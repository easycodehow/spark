 PWA: 웹과 앱의 장점을 합친 기술
PWA(Progressive Web App)는 웹 기술로 만들지만 앱처럼 동작하는 새로운 형태의 서비스입니다.

📲
앱처럼 설치
웹사이트를 홈 화면에 추가하여 앱처럼 사용할 수 있음

📡
오프라인 동작
인터넷 연결이 없어도 기본 기능 사용 가능

🔔
푸시 알림
앱처럼 푸시 알림을 보낼 수 있음

PWA 실제 사례
트위터: 앱 대신 PWA 버전을 주력으로 사용
스타벅스: PWA로 주문과 결제 시스템 구현
핀터레스트: PWA 도입 후 사용자 참여도 60% 증가
인스타그램: 인터넷이 느린 지역을 위한 PWA 버전 제공

📱 3가지 기능 모두 넣기 좋은 앱들:
⭐ 1. 개인 메모장 (가장 추천)
기능:
- 메모 작성/수정/삭제
- 다크모드 (눈 편함)
- 메모 복사 (다른 곳에 붙여넣기)
- 카톡/문자로 메모 공유
- 검색 기능


⭐ 2. D-Day 카운터 (학생들이 좋아함)
기능:
- 중요한 날짜 등록 (시험, 생일, 방학)
- 남은 일수 표시
- 다크모드
- 친구에게 D-Day 공유 (카톡)
- D-Day 텍스트 복사


⭐ 3. 링크 북마크 앱
기능:
- 유용한 링크 저장 (유튜브, 사이트 등)
- 카테고리 분류
- 다크모드
- 링크 복사 (빠른 접근)
- 친구에게 링크 공유


⭐ 4. 명언/좋은 글 모음
기능:
- 좋아하는 명언 저장
- 다크모드
- 명언 복사 (SNS에 쓰기)
- 친구에게 공유
- 랜덤 명언 보기


⭐ 5. 공부 기록장
기능:
- 오늘 공부한 내용 기록
- 과목별 분류
- 다크모드
- 공부 내용 복사
- 친구/부모님께 공유
